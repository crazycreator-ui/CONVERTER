    Frequently Asked Questions

    Is there a limit to how many PDF files I can merge?

    You can merge up to 20 PDF files at once. If you need to merge more, you can merge them in batches and then combine the resulting PDFs.

    What's the maximum file size allowed?

    Each individual file can be up to 50MB, and the total size of all files cannot exceed 100MB. For larger files, consider compressing them first.

    Are my files secure when using this tool?

    Yes, all file processing happens in your browser. Your files are not uploaded to our server, ensuring complete privacy and security. All files are automatically deleted after processing.

    Will the quality of my PDFs be affected when merging?

    No, our tool maintains the original quality of your PDF files. The merged PDF will have the same quality as the original files.

    Can I merge password-protected PDF files?

    No, password-protected PDFs cannot be merged with this tool. You'll need to remove the password protection first using our "Unlock PDF" tool, then merge the files.
